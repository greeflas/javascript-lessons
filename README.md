WebStorm settings
-----------------

### Live templates

`File` -> `Settings...` -> `Editor` -> `Live templates` -> `+ Live template`

1. Abbreviation `cl`

2. Template text `console.log($END$);`

3. Define `JavaScript`

4. Click to `Apply` and `OK`

### JavaScript standard

`File` -> `Settings` -> `Languages & Frameworks` -> `JavaScript` -> `JavaScript language version` -> `ECMAScript 5.1`
